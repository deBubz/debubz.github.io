# Building Vue WebApp

To finish up everything, all I need is to build a WebApp/ User Interface to interact with the data. To beter fit my wireframe design in the previous week, I added another collection called "bDeviceDetails" into the database. This collection is to store the individual details of my devices if there are multiple active devices in the system. Since I only have 1 active device at the time, I added some sample data to test while building the site.

I begun with building the back end processes of the webapp (as shown in the tutorials I completed last week), the API which will be used to take data from the database to be presented on the webapp. Overall, my api have the ability to:

  • return the details of all devices
  • show the details of a single device (searched by using the UniqueId of the device)
  • show the data of a single device (searched by using the UniqueId of the device)
  • the ability to toggle the status of a single device( OK to BAD, and vice versa)
